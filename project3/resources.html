Questions & Answers

A. Which element of the site is the graphic created from scratch?

The graphic that I made from scratch is the logo that is the banner on top of each page. I created it using Photoshop.

B. What type of embedded media was used on the site and on which page(s) it is located?

I used an embedded YouTube video on the Welcome page that demonstrates what New Student Orientation at RIT is all about.

C. Which image on the site uses a CSS image rollover effect?

The image used on the Clubs page uses an effect that displays a caption when the image is hovered over. I followed a tutorial from Tutorialspoint to achieve this and changed a few things like the position and color of the text.

D. Provide details about any JavaScript used and an explanation of why that script was selected.

I found a Javascript script on W3Schools that creates a photo slideshow. I implemented this on the Tips & Attractions page because it lets me display photos of all the great attractions there are to see in Rochester. By using the slideshow, it keeps the page clean and clutter-free because there aren't several photos that needs to be scrolled through.

E. Provide a 2-3 paragraph summary of the website. What 3 factors of the site were done really well and 2 factors that could use improvement.

The website is a great informational source for Freshmen students entering RIT. It has important information that is summarized and easy to understand. It can be hard to find this information on the actual RIT website so my website does a good job at presenting it. I believe the factors that I have done really well on the site are the content, photos, and logo. The content presents RIT freshmen with a load of information to settle into the RIT community. The photos add more details and make the site more interesting. The logo resembles the tiger stripes and makes the site stand out really well.

There are some factors that could use improvement as well. If I had more time, I feel that I could have spent more time on the design. I could have made the site look less "basic." I could have enhanced the navigation bar and used JavaScript to add more features. Another thing is I could have improved on is the media queries. It does not look as polished as the desktop site, I would have liked to add some enhancements to make it more mobile-friendly.
